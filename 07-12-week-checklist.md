# 12-Week Day-by-Day Checklist (Working Professional Friendly)

This checklist converts the roadmap into **daily executable actions**.

**Time assumptions:**
- Mon–Fri: ~2 hours/night
- Saturday: 2–3 hours (morning only)
- Sunday: Optional / review

If a day is missed, **continue forward**. Do not try to catch up aggressively.

---

## WEEK 1 – Docker Foundations

**Mon:** What is Docker, images vs containers, Docker architecture  
**Tue:** Install Docker, run first containers, understand Docker CLI  
**Wed:** Dockerfile basics, build your first image  
**Thu:** Multi-stage builds, image optimization concepts  
**Fri:** Use AI to refactor Dockerfile, read one Docker best-practice blog  
**Sat:** Dockerize a simple Spring Boot app  
**Sun (optional):** Review notes
